# Visión de producto

## Qué es

WIAMarketingOS es un sistema operativo interno para crear, lanzar, medir y optimizar campañas comerciales.

Debe permitir pasar de una idea de campaña a un flujo completo:

```text
idea de campaña
brief
landing o pantalla generada
formulario
lead
empresa/contacto
oportunidad
seguimiento comercial
automatización
métrica de conversión
aprendizaje para la siguiente campaña
```

## Qué no es

WIAMarketingOS no es:

- Un CRM clínico.
- Un sistema para pacientes.
- Un repositorio por campaña.
- Una copia completa de HubSpot desde el primer día.
- Un editor visual libre tipo Webflow construido desde cero.

## Principios

1. **Separación radical de producto y captación**
   El CRM que vendemos a clientes vive fuera. Este sistema existe para vender mejor.

2. **Campañas como activos reutilizables**
   Dental, estética o fisioterapia no deben ser repos distintos. Deben ser campañas dentro del mismo sistema.

3. **Generación controlada, no caos visual**
   La IA no debe escribir páginas arbitrarias. Debe devolver estructuras validadas que nuestro sistema renderiza con bloques seguros.

4. **Medición desde el inicio**
   Cada landing, formulario y CTA debe generar eventos. Sin eventos no hay aprendizaje.

5. **Consentimiento y trazabilidad**
   Cada lead debe guardar fuente, campaña, consentimiento, fecha, IP o metadatos razonables y política aceptada.

6. **Venta asistida, no automatización ciega**
   Primero preparar mensajes, priorizar leads y crear tareas. Automatizar envíos reales solo cuando el proceso esté validado.

## Usuarios internos

- Dirección: ve rendimiento, oportunidades y pipeline.
- Marketing: crea campañas, pantallas, variantes y formularios.
- Ventas: gestiona leads, llamadas, demos, pilotos y cierres.
- Operaciones: revisa automatizaciones, integraciones y calidad de datos.

## Métrica norte

La métrica principal no debe ser "número de landings creadas".

La métrica principal debe ser:

```text
oportunidades cualificadas generadas por campaña y coste por oportunidad
```

Métricas secundarias:

- Conversión de visita a lead.
- Conversión de lead a demo.
- Conversión de demo a piloto.
- Conversión de piloto a cliente.
- Tiempo de primera respuesta.
- Porcentaje de leads sin seguimiento.
- Valor estimado de pipeline por campaña.
